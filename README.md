OOTD-Diary is an iOS project for photo sharing and management.

  - Upload their photo to this iOS app.
  - View the new photos they have uploaded
  - More features...

### Version
1.0.0

### Installation

```sh
$ git clone [git-repo-url]
$ cd OOTD-Diary
$ open OOTD diary.xcodeproj
```
